# Monthly Retainer Pitch — Scripts

## For Byron (Furry Furrows)

> Hey Byron, quick thing — I'm moving your website over to better hosting this week so it runs faster and I can manage it more easily.
>
> Since the site's live and I built it, I wanted to set up a simple monthly arrangement: **$50-75/month** covers hosting, any small updates you need, and basic support if anything breaks.
>
> That way you're not stuck if something goes wrong, and I can keep it maintained properly. Does that work for you?

## For Ellie (After Deliverables)

> Hey Ellie, now that [deliverables] are wrapped up, I wanted to talk about what ongoing support looks like.
>
> You've got the CRM, the funnel, the content system — but these things need maintenance. Updates break, platforms change, and you'll want tweaks as the business evolves.
>
> I'm thinking **$200-300/month** for ongoing support: CRM maintenance, content system updates, light IT help, and priority access when something urgent comes up.
>
> This way you're not starting from scratch every time something needs attention. Want to try it for a few months and see how it goes?

---

## Key Principles

1. **Frame it as protection**, not a charge
   - "So you're not stuck if something breaks"
   - "So you're not starting from scratch"

2. **Keep it simple**
   - Don't over-explain
   - Don't list 15 services
   - Just: hosting + updates + support

3. **Make it easy to say yes**
   - Low monthly amount
   - "Try it for a few months"
   - No long commitment

4. **Connect it to existing work**
   - "Since I built it..."
   - "Now that [X] is done..."
