## BUILD SYSTEM (Key Rules)

**Design standard:** Google Antigravity — gradient mesh backgrounds, floating SVG accents at low opacity, bold oversized type, generous whitespace, scroll-triggered animations (fade-up, 40px, 0.6s), hover effects on all CTAs.

**Zooted Farms OVERRIDE:** Dark editorial, NO gradient mesh, Syne + Bebas Neue, aggressive motion, cultural/raw copy tone.

**Font map:**
- Real estate / legal → Inter + Playfair Display
- Trades → Inter + Space Grotesk
- Coaching / education → DM Sans + Lora
- Streetwear / creative → Syne + Bebas Neue
- Health / pet → Nunito + Merriweather

**6 phases:** Intake → Blueprint → Lovable Prompt Sequence → Copy → Launch Checklist → Client Handoff Doc

**Workflow triggers:**
- "Research Mode" → find UI repos for client industry
- "Build Mode" → generate full Lovable prompt sequence, copy-paste ready
- "Ad Mode" → Hormozi landing page spec + Google Ads copy set
- "Debrief" → lessons learned + proposed system update
- "System Audit" → review all past builds for patterns
