# GIVANS GLOBAL — HERO SECTION BOILERPLATE

## Core Boilerplate (inject into every hero prompt)
"Hero section: min-height 100vh, flexbox column justify-center 
align-items flex-start, text block max-width 580px left-aligned, 
no absolute positioning on text, z-index layering so text never 
overlaps nav (nav z-index: 50, hero content z-index: 10). 
Entrance animation: text block fades up 40px over 0.6s on load. 
CTA button: large, high-contrast, subtle scale hover effect (1.03). 
Mobile: text stacks full width, font scales down gracefully 
(clamp 32px–64px), CTA always above fold at 375px, min-height 
48px on all tap targets."

---

## Background Variant A — Gradient Mesh (default)
"Background: CSS radial-gradient mesh using two brand colors at 
0.15 opacity on dark base (#0A0A0A or #0D0D1A). 
Add 2–3 floating SVG circles/blobs at 0.08 opacity, 
different sizes, positioned behind text block and to the right. 
Apply subtle CSS animation: float up/down 6px over 4s ease-in-out 
infinite on the SVG shapes."

## Background Variant B — Dark Image Overlay
"Background: full-bleed image with a dark gradient overlay 
(linear-gradient from rgba(0,0,0,0.75) left to rgba(0,0,0,0.3) 
right). Image must not compete with text — ensure text contrast 
ratio is minimum 4.5:1. Add a subtle grain texture overlay 
at 0.04 opacity for depth."

## Background Variant C — Abstract Geometric (Zooted Farms)
"Background: solid dark (#0A0A0A). Add large abstract geometric 
shapes (triangles, irregular polygons) in brand accent color at 
0.06–0.10 opacity, clipped to right side of viewport. 
Apply stagger entrance animation on shapes (delay 0.1s per shape). 
Typography is the hero — no image, no mesh."

---

## Use Guidelines
- Gradient Mesh: Real estate, legal, professional services, 
  pressure washing, courses
- Dark Image Overlay: When client has a strong brand photo 
  or before/after result image
- Abstract Geometric: Zooted Farms, creative brands only
- Hormozi Ad Landing Pages: NONE of the above — white or 
  deep dark flat background only, no decoration
