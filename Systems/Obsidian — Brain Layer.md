# Obsidian — Brain Layer

> Obsidian is my thinking environment and long-term memory. It's not a task manager or a calendar — it's where ideas become decisions and decisions become systems.

---

## What Goes Here

- **Ideas** — raw captures, half-formed thoughts, things to explore
- **Project notes** — goals, scope, decisions, progress logs for every active project
- **System docs** — how my tools work, why I set things up the way I did
- **Reference** — things I'll want to look up later (not a Wikipedia, just what's personally useful)
- **Meeting/session notes** — what was decided, not just what was discussed
- **AI session logs** — key outputs and decisions from Claude Code sessions

## What Does NOT Go Here

- Tasks with deadlines (use a dedicated task tool)
- Full code (use the repo; link to it from here)
- Duplicating things that live in other systems — just link or reference them

---

## Note Types I Use

| Type | Purpose | Example |
|------|---------|---------|
| `Project — [Name]` | One note per project | `Project — Client Site Redesign` |
| `System — [Name]` | Documents a tool or process I use | `System — Daily Review` |
| `Idea — [Name]` | Raw capture, unprocessed | `Idea — AI Funnel Builder` |
| `Reference — [Name]` | Something I'll look up again | `Reference — Tailwind Cheatsheet` |
| `Log — [Date]` | Session or decision log | `Log — 2026-03-23` |

---

## Folder Structure

```
Obsidian Vault/
├── Systems/          ← how my tools and processes work (you are here)
├── Workflows/        ← step-by-step processes for recurring work
├── Projects/         ← one folder or note per project
├── Ideas/            ← raw captures (process weekly)
├── Reference/        ← lookup notes
└── Logs/             ← session notes, decision logs
```

---

## How I Use Obsidian With Claude Code

- Before a Claude Code session: write a brief project note with the goal and context
- During: paste key decisions or outputs back into Obsidian as a session log
- After: update the project note with what changed, what's next

This keeps future sessions from starting cold.
